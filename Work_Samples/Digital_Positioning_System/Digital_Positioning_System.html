DPS is a system that uses MCU, ADC, and EEPROM to produce a precision location of actuation on an independent linearity based linear potentiometer

MCU takes readings from the potentiometer using the ADC

These ADC values are then put through a correction algorithm that uses EEPROM to produce an accurate distance reading based on the point of actuation
